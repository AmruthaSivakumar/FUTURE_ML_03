Project Title: Resume Screening System

Description:
This project builds an ML-based resume screening system that ranks candidates based on job role requirements.

Tools Used:

Python

Pandas

Scikit-learn

spaCy / NLTK

Jupyter Notebook

Steps Performed:

Resume text extraction and cleaning

Skill matching with job description

Candidate ranking

Skill gap analysis

Outcome:
The system ranks resumes and highlights missing required skills
